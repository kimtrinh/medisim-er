Engine Settings

Engine cascade: every order tries ① Gemini (free, fast) first; if it's rate-limited, out of daily quota, or down, the turn automatically reruns on ② your Claude API key, then on ③ a local engine (bridge or LM Studio) — and the sim climbs back to ① at the next case. A hung engine times out and cascades instead of freezing the sim.

→ Get a free Gemini key (aistudio.google.com/apikey · takes ~1 min). Free tier is ~20 requests/day (set the budget below); the cascade skips Gemini automatically once it's spent. Also powers HD character voices. Stored only in this browser's local storage.

→ Get a Claude API key (console.anthropic.com). Pay-per-use, no daily cap, ~2–6s/turn. Takes over the moment Gemini fails so the case never stalls. Stored only in this browser's local storage; never sent anywhere but api.anthropic.com.

Tried automatically when ① and ② both fail: first the Claude subscription bridge (http://localhost:8787/v1, $0 extra, needs a one-time claude /login), then the LM Studio/Ollama server at the URL above.

LM Studio setup: Developer tab → Start Server (port 1234) → load your Qwen model → enable CORS in server settings (required so this page can reach it). Then click Detect above.
No key yet? A Gemini key is free — get one here and paste it into ① above. Your key is stored only in this browser and calls the provider directly; this site never sees it and never pays for your usage.
